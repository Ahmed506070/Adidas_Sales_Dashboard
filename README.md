# Adidas Sales Dashboard

This project analyzes Adidas sales data using **Python** for data cleaning and **Power BI** for visualization.

## 📊 Dashboard Highlights
- Total Sales: 899.90M
- Total Units Sold: 2M
- Total Profit: 332.13M

## 🗺️ What's Inside
- Sales by Region (West, Northeast, Southeast, South, Midwest)
- Units Sold by Product
- Operating Profit by Month
- Sales Distribution by State on Map

## 🛠️ Tools Used
- **Python** (Jupyter Notebook) - Data Cleaning
- **Power BI** - Data Visualization
